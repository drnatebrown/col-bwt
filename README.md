# col-BWT
Alpha Version 0.2.0

# How-to
### Download and Compile

Clone and Compile:
```console
git clone https://github.com/drnatebrown/r-index-f.git
cd r-index-f

mkdir build && cd build
cmake ..
make
```

### Commands

All functionality provided by:
```console
./col-bwt
```

# Dependencies
- Requires python3 to run col-bwt command
